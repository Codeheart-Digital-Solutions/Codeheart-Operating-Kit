## Placement

Reusable Operating Kit doctrine belongs in this repository. Consumer-specific product guidance,
plans, memory state, credentials, local approvals, and environment details belong in the consumer
repository or workspace.

Use `docs/repo/reference/placement-contract.md` before introducing new durable paths.
