
strongest practical main-thread proxy result:

- The durable mechanics should live under the owner selected by routing, such as the owning route,
  module, package, product, repository, or source-area script boundary.

  narrow operation can be deterministic after routing and approvals.

  as a precondition.

  real reuse.

  ambiguity questions, fallback choices, and consequences.

  if those are not already known.
