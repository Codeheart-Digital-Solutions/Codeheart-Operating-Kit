# Local User Layer

This folder is for ignored local user preferences and notes. It is not managed Operating Kit
content and should not contain secrets, credentials, tokens, or public release material.

Keep shared non-secret consumer configuration in `.codeheart/kit.config.yaml`. Keep installed
version, checksums, and update-check state in `.codeheart/kit.lock.yaml`.
