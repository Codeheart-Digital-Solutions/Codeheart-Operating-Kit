# Maintain Documentation Indexes

Use this runbook when discoverability changes.

## Procedure

1. Identify the nearest README router.
2. Identify parent indexes that route to the changed area.
3. Add concise entries for new durable docs.
4. Update paths after moves or renames.
5. Remove current-routing entries for archived or deleted docs.
6. Keep historical references in completed plans when they are traceability records.
7. Validate that links and paths are current.
