# Planning Workflows

This managed domain owns reusable planning workflows for discovery, implementation planning,
implementation execution, and planning document review.

## Routes

- Discovery workflow: `runbooks/discovery-workflow.md`
- Draft implementation plan: `runbooks/draft-implementation-plan.md`
- Execute implementation plan: `runbooks/execute-implementation-plan.md`
- Review planning document: `runbooks/review-planning-document.md`
- Planning document lifecycle: `reference/planning-document-lifecycle.md`

## Ownership

These files are Operating Kit managed content. Consumers should add repository-specific plan
details in their own `docs/repo/`, product docs, or module docs, not by editing managed files under
`.codeheart/kit/docs/planning-workflows/`.
